**Branch**: `001-khaos-kart-game` | **Date**: 2026-03-07 | **Spec**: [spec.md](./spec.md)

## Summary

KHAOS KART is a full HTML5 kart racing game with cell shading visuals, 8 unique characters, 5 themed circuits, an item system with 25 items, local split-screen (2-4 players), asymmetric crew mode (driver/gunner), online multiplayer rooms (8 players), dynamic weather, AI opponents, mobile touch controls, and PWA support. Built with Three.js + Rapier.js + Socket.io + Vite, using a server-authoritative multiplayer model with client-side prediction.

## Technical Context

**Language/Version**: JavaScript (ES2022+), GLSL for shaders
**Primary Dependencies**: Three.js (r182+), Rapier.js (WASM), Socket.io, Howler.js, nipple.js, vite-plugin-pwa
**Storage**: localStorage (player profiles, settings, ghost replays, records)
**Testing**: Playwright (E2E, multi-browser, mobile simulation)
**Target Platform**: Modern browsers (Chrome, Firefox, Safari) on desktop and mobile
**Project Type**: Web application (client game + lightweight Node.js game server)
**Performance Goals**: 60 fps desktop, 30 fps mobile, <8s load time, <100ms network perceived lag
**Constraints**: No installation required (browser-only), offline-capable for solo modes (PWA), <256 kbps bandwidth per client
**Scale/Scope**: 8 concurrent players per room, 8 characters, 5 circuits, 25 items, 3 AI difficulty levels

**Structure Decision**: Client-server web application. The client (src/) is a Vite-built SPA with Three.js game rendering. The server (server/) is a lightweight Node.js process handling authoritative multiplayer state. Both share no code at build time but follow the same data model contracts.
