# Quickstart: KHAOS KART

**Branch**: `001-khaos-kart-game` | **Date**: 2026-03-07

## Prerequisites

- Node.js 20+
- npm 10+

## Setup

```bash
# Clone and install
git clone <repo-url>
cd kart-toons
npm install

# Start development server (client)
npm run dev
# → Opens at http://localhost:5173

# Start game server (multiplayer, separate terminal)
npm run server:dev
# → WebSocket server at ws://localhost:3000
```

## Key Commands

```bash
npm run dev              # Start Vite dev server (client)
npm run build            # Production build
npm run preview          # Preview production build
npm run server:dev       # Start game server (dev, with auto-reload)
npm run server:start     # Start game server (production)
npm run test:e2e         # Run Playwright E2E tests
npm run test:e2e:ui      # Run Playwright with UI
```

## Tech Stack Summary

| Layer | Technology | Purpose |
| ----- | ---------- | ------- |
| Rendering | Three.js (r182+) | 3D WebGL rendering, toon shading |
| Physics | Rapier.js (WASM) | Collision detection, raycasting |
| Kart movement | Custom code | Arcade steering, drift, boost |
| Networking | Socket.io | Client-server WebSocket communication |
| Server | Node.js | Authoritative game server |
| Audio | Howler.js + Web Audio API | SFX, dynamic music stems, engine synthesis |
| Touch controls | nipple.js | Mobile virtual joystick |
| Build | Vite.js | Development and production builds |
| PWA | vite-plugin-pwa | Service worker, offline support |
| Testing | Playwright | E2E automated tests |
